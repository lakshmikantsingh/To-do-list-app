# To-Do List App

A modern, responsive To-Do List web application built with HTML, CSS (Bootstrap), and JavaScript.

## Features
- Add, complete, and delete tasks
- Task categories and priorities
- Sidebar navigation with icons
- Sticky header and form for quick task entry
- Responsive design for all devices
- Local storage support (tasks persist after refresh)

## Getting Started
1. Clone or download this repository.
2. Open `index.htm` in your browser.

## File Structure
- `index.htm` — Main HTML file
- `style.css` — Custom and Bootstrap-based styles
- `script.js` — App logic (add, complete, delete, filter tasks)
- `README.md` — Project documentation

## Customization
- Edit `style.css` for UI/UX changes
- Update `script.js` for new features or logic
